## 🔧 项目经验

### **C++ 基于 Protobuf 的发布订阅式消息队列**  
*2026.01 ~ 2026.03 | 自主开发*

> 仿 RabbitMQ 实现一个轻量级消息队列组件，支持跨主机的消息发布与订阅，具备可靠的消息推送机制。

- 基于 AMQP 模型设计，实现完整的消息队列核心逻辑（Exchange/Queue/Binding）
- 使用 SQLite 轻量数据库进行数据持久化，保障系统宕机后的恢复能力
- 应用层基于 Protobuf 设计自定义协议接口，解决网络传输中的序列化问题
- 使用 gtest 框架完成单元测试，覆盖存储引擎、路由算法等核心模块
- 基于 Muduo 网络库实现高性能 TCP 通信服务端与客户端，掌握 Epoll 事件驱动模型
- 实现基于应用层 ACK 的消息确认机制，确保消息不丢失
